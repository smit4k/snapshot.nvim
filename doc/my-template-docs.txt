snapshot.nvimCreate beautiful screenshots of your code, directly inside Neovim
FEATURES                                           *my-template-docs-features*

- Code snapshots with syntax highlighting
- Customizable settings (line numbers, padding, font, etc.)
- Blazingly fast Rust renderer


INSTALLATION                                   *my-template-docs-installation*

**Prerequisites**: Neovim >= 0.9.0, Rust & Cargo
<https://rust-lang.org/tools/install/>

Use your favorite plugin manager!

Ex: using lazy.nvim

>lua
    return {
      "smit4k/snapshot.nvim",
      build = "cd generator && cargo build --release",
      config = function()
        require("snapshot").setup({
            padding = 80,
            line_height = 28,
            font_size = 20,
            background = "#282c34",
            shadow = true,
            line_numbers = false,
            start_line = 1,
      )}
      end,
    }
<


USAGE                                                 *my-template-docs-usage*

Select code in Visual mode and run `:Snapshot`. The snapshot saves to
`~/snapshot-{timestamp}.png`
