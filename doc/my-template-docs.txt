FEATURES                                           *my-template-docs-features*

- Code snapshots with syntax highlighting
- Customizable settings (line numbers, padding, shadow, render scale, etc.)
- Blazingly fast Rust renderer


INSTALLATION                                   *my-template-docs-installation*

**Prerequisites**: Neovim >= 0.9.0

Use your favorite plugin manager!

Ex: using lazy.nvim

>lua
    return {
      "smit4k/snapshot.nvim",
      config = function()
        require("snapshot").setup({
            -- Optional config settings, defaults shown below
            snapshot_dir = "~/Pictures/snapshots", -- Directory where snapshot images are saved to
            padding = 25,
            line_height = 28,
            font_size = 24,
            shadow = true,
            line_numbers = false,
            start_line = 1,
            border_radius = 5,
            outer_background = "#ffffff" -- use #00000000 for transparent
            outer_padding = 15,
        })
      end,
    }
<


USAGE                                                 *my-template-docs-usage*

Select code in Visual mode and run `:Snapshot`. To capture an entire file, just
run `:Snapshot` in Normal mode.
